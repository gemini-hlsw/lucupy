    Group dataclass

    Bases: ABC

    This is the base implementation of AND / OR Groups. Python does not allow classes to self-reference unless in static contexts, so we make a very simple base class to self-reference from subclasses since we need this functionality to allow for group nesting.

    Attributes:

    Name Type Description
    id GroupID

    the identification of the group.
    group_name str

    a human-readable name of the group.
    number_to_observe int

    the number of children in the group that must be observed for the group to be considered complete.
    delay_min(timedelta) int

    used in cadences.
    delay_ma(timedelta) int

    used in cadences.
    children Union[List[Group], Observation]

    member(s) of the group
